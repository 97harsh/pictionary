# **App Name**: PictoGame Master

## Core Features:

- Random Word Generation: Generate a random word from the Animals, Objects, Actions, Movies, Places, and Food categories.
- Word Display: Display the generated word to the current drawer.
- Word Skipping: Allow the drawer to skip a word within a configurable limit per round.
- Timer Management: Manage the round timer with configurable durations (30s, 60s, 90s).
- Team and Turn Management: Manage teams, turns, and the current drawer.
- Scoreboard Tracking: Display and track scores for all teams, including score history.
- Game Reset: Allow resetting the game with a 'Reset game' button.

## Style Guidelines:

- Primary color: Dark indigo (#3F51B5) to establish a focused, game-master-like feel.
- Background color: Very dark indigo (#263272) for a high-contrast, dark-mode-first approach.
- Accent color: Electric purple (#8A2BE2) to highlight active elements and interactive buttons.
- Body and headline font: 'Inter', a sans-serif for clean readability and a modern feel.
- Use simple, geometric icons to represent different word categories.
- Mobile-first responsive design with big, easy-to-tap buttons.
- Subtle animations for timer countdown and score updates.
